引用 5 楼 wanby 的回复:> cmd

::打开命令提示符窗口 cd PATH ::你的文件夹所在目录 for /r %i in () do if not exist %~dicomplete%~ni md %~dicomplete%~ni else md %~dicomplete%~ni(%~xi) ::在文件夹所在盘符根目录建立complete文件夹，并为每一个文件建立一个同名文件夹,当出现两个文件名相同而扩展名不同时，建立的文件夹名称以括号+扩展名结束。 如果建立完了还想把每个文件对应拷贝到同名文件夹时，用下面的命令 for /r %i in () do if not exist %~dicomplete%~ni (md %~dicomplete%~ni & copy %~nxi %~dicomplete%~ni) else md %~dicomplete%~ni(%~xi) & copy %~nxi %~dicomplete%~ni(%~xi)

PS：以上命令有文件名以两个以上点号加后缀结束的情况下不可用，需进一步改进，改进方法另议。
