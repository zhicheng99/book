# Mac下mysql配置文件没有my-default.cnf，无法配置my.cnf

# Mysql不提供my-default.cnf文件的版本

官网说：从5.7.18开始不在二进制包中提供my-default.cnf文件。

# 针对用户

针对mac端，并且没有my-default.cnf的用户

# 添加并修改my.cnf文件

修改期间必须关闭mysql

我们要在/etc下新建my.cnf

$ cd /etc 
$ sudo vim my.cnf
